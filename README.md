## 使用技術
### HTML
- wrapperを最小限に抑えた冗長性のないコード
- セマンティックタグを使用し、可読性向上
- 全体的に名前空間化されたクラス名
  途中の設計変更でクラス名のルールが統一できていない箇所もありますが、改善点として認識しています。

### CSS
- grid,flexを活用したレスポンシブルレイアウト
- ナビゲーションとハンバーガーメニューを兼用した設計
- カスタム変数でJSと連携(ハンバーガー)

### JS
カルーセル、ハンバーガー、モーダルの3箇所にJSを使用しています。

純粋に、クラス付け替えやカスタムプロパティを実装しています。

### Photoshop/illustrator
#### カルーセルバナー
- 1枚目:写真ACの画像を2枚合成
- 2枚目:AdobeFireflyでAI生成
- 3枚目:illustratorで自作
#### アイコン/ロゴ
- SVGはillustltorで制作、手動で一部編集
- Web向けに軽量化
