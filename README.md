# 桜山町
※架空の自治体です。現実の地名とは一切関係ありません。

大都市近郊の自然豊かな地域。
休日には観光客が多く訪れます。

町名にこだわりはありません。制作実績用の無難な名前にしました。

## 使用技術
### HTML
セマンティックタグを使用。wrapperを最小限に抑えた冗長性のないコードという設計思想でコーディングしています。

全体的に名前空間化されたクラス名を目指しました。途中の設計変更でルールが統一できていない箇所もあります。

### CSS
grid,flexを活用したレスポンシブルレイアウトを採用しました。

navをナビゲーションとハンバーガーメニューでレスポンシブルに兼用した設計ですが、どうしても拡張性が低くなってしまうことは認識しています。

カスタム変数でJSと連携し、ハンバーガーのアニメーションにタイムラグをつけました。

### JS
カルーセル、ハンバーガー、モーダルの3箇所にJSを使用しています。

### Photoshop/illustrator
#### カルーセルバナー
1枚目は写真ACさんの画像を2枚合成しました。

2枚目(市役所)はAdobeFireflyで生成。

3枚目はイラレで自作しました。
#### アイコン/ロゴ
イラレで作成し、SVGを一部編集しています。
